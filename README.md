# 文豪迷犬怪奇潭 - 弹珠 RPG

🎮 基于《文豪野犬》IP 的弹珠 RPG 游戏复刻版

## 游戏特色

- **弹珠战斗系统** - 拉弹弓发射角色，利用物理碰撞击败敌人
- **角色收集** - 抽卡系统获取文豪角色（SSR/SR/R）
- **关卡推进** - 3 大章节 15 个关卡，逐步解锁
- **资源管理** - 体力、宝石、金币系统
- **队伍组建** - 最多 4 人小队

## 角色列表

### SSR
- 太宰治（暗/平衡）- 技能：人间失格
- 中原中也（暗/攻击）- 技能：污浊了的忧伤之中
- 芥川龙之介（暗/攻击）- 技能：罗生门
- 中岛敦（光/平衡）- 技能：月下兽
- 国木田独步（木/技术）- 技能：独步吟客

### SR
- 泉镜花、谷崎润一郎、宫泽贤治、与谢野晶子、江户川乱步

### R
- 立原道造、梶井基次郎、佐佐城信子、梦野久作、广津柳浪

## 玩法说明

1. **开始游戏** - 点击主线剧情选择关卡
2. **组建队伍** - 在角色界面点击角色加入队伍（最多 4 人）
3. **战斗** - 拖拽角色发射，利用物理碰撞击败敌人
4. **抽卡** - 使用宝石召唤新角色（单次 50 宝石，十连 500 宝石保底 SR+）

## 本地运行

直接用浏览器打开 `index.html` 即可游玩。

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 渲染战斗场景
- LocalStorage 保存游戏进度
